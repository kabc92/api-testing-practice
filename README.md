# API TEST AUTOMATION FRAMEWORK 

API Test automation built, RestAssured and JUnit5

## TECH STACK 
- Java 21
- Rest Assured 6.0.0
- JUnit Jupiter 5.11.4
- Maven 3.9.16

## TEST COVERAGE
- GET, PUT, POST, PATCH, DELETE requests
- Response body validation wth JsonPath
- Header Validation
- Status code validation
- Negative test cases
- API Key Authentication
- Request Specification with RequestSpecBuilder

## HOW TO RUN
```bash
mvn test
```
## APIs Tested
- JSONPlaceholder (https://jsonplaceholder.typicode.com)
- PetStore Swagger (https://petstore.swagger.io)
